# Canadian Health Measures Survey (CHMS) Cycle 4 Medications

This dummy data representing the medication portion of the third cycle
of the Canadian Health Measures Survey (CHMS). The CHMS survey is
conducted by Statistics Canada.

## Usage

``` r
data(cycle4_meds)
```

## Source

Statistics Canada
